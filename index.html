Lesson Learned


Konsistensi dalam Penamaan Kelas dan ID

Memastikan konsistensi dalam penamaan kelas dan ID membantu dalam pemeliharaan kode CSS dan HTML.


Pemisahan Struktur dan Gaya

Memisahkan struktur HTML dan gaya CSS memungkinkan untuk mengubah tampilan situs dengan lebih mudah dan meningkatkan keterbacaan kode.


Responsif dan Fleksibilitas

Menggunakan teknik responsif seperti media queries dan flexbox memastikan bahwa situs web dapat diakses dengan baik di berbagai perangkat dan layar.


Pentingnya Validasi

Memvalidasi kode HTML dan CSS membantu memastikan bahwa situs web dapat diakses dengan baik oleh berbagai browser dan meminimalkan kemungkinan bug.


Konsistensi Warna dan Tampilan

Menggunakan palet warna yang konsisten dan tampilan yang seragam membantu dalam menciptakan pengalaman pengguna yang menyatu dan profesional.
